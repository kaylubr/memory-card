# Dunder Mifflin Memory Match

Dunder Mifflin Memory Match is a fan-made memory card game inspired by *The Office*. Players flip cards and try to not to find its matching pairs while keeping track of their moves.

## Features

* Memory card matching gameplay
* Responsive layout
* Move tracking
* Card-flip animations
* Office-inspired visual design
* Accessible button and keyboard interactions
* Support for reduced-motion preferences

## Technologies

* React
* TypeScript
* CSS
* Vite

## Getting Started

Clone the repository:

```bash
git clone https://github.com/kaylubr/dunder-mifflin-memory-match.git
```

Open the project directory:

```bash
cd memory-game
```

Install the dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the local URL shown in your terminal to view the application.

## How to Play

Select a card, then select another card.

When they do not match, points get added. When both cards match, the game resets

Continue until every pair has not been found.
